# For Users

## Опис

Це IT сервіс. Він надає можливість організовувати спільні дискусії та ігри в публічних закладах. Наприклад в кав'ярнях, кафе, ресторанах, піцеріях, книгарнях та ін.
Сервіс безоплатний.
